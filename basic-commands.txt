===Basic command==https://github.com/freeitgroupe/git-lessons/blob/master/basic-commands.txt
git init - инициализация
git status - статус проекта
git add . - добавление всех изменненых файлов для коммита
git commit -m "name-commit" - коммит
git remote add origin https://github.com/freeitgroupe/js_lessons.git - определяем репозиторий
git push -u origin master - загружаем проект на гит
git clone https://xxxxxxx@bitbucket.org/xxxxa/xxxxxx.git -b developers --single-branch - клон из ветки
git pull origin developers - подгрузка изменений из ветки developers
git pull https://github.com/freeitgroupe/qa_crm main - подгрузка изменений из проекта ветка main
